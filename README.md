### Project Structure
```
frontend/  - HTML + Tailwind + Firebase JS SDK
backend/   - Python Flask + Firebase Admin + OpenAI
docs/      - Architecture, Demo script
ai_prompts/ - Exact prompts used in ChatGPT/Claude/Cursor
```

### How to Run
1. Backend: `cd backend && pip install -r requirements.txt && python app.py`
2. Frontend: Open `frontend/index.html` or deploy to Vercel
3. Add Firebase config + OpenAI key in .env
